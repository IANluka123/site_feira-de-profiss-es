Projeto Sensor de Umidade

O sensor de umidade tem o objetivo de mostrar para a criadora da suculenta como está a umidade do solo.

Quando o solo precisa ser regado, ele acende o led vermelho, se o solo está com boa umidade acende o led verde. Ele funciona a partir da programação no computador, que calcula essa quantidade de umidade.

Este estante esta com parceria com o de Biotecnologia
